Đề án Sprint Plan

Lộ trình triển khai (Sprint Plan)

Data-Driven Management System

Sprint 1

Giai đoạn 1: Xây dựng "Xương sống" dữ liệu

Task 1.1: Thiết kế Master Sheet (Database)

Tạo file Google Sheet với các Tab: GiangVien, LopHoc, SinhVien, DiemDanh, TienDo_Lab_Quiz.

Task 1.2: Viết Apps Script định nghĩa hàm CRUD

Xây dựng API cơ bản để Thêm/Sửa/Xóa dữ liệu trực tiếp từ Sheet.

Task 1.3: Thiết lập AppSheet Dashboard cơ bản

Tạo giao diện nhập liệu đầu tiên cho Giảng viên trên AppSheet.

Sprint 2

Giai đoạn 2: Tự động hóa Drive & Chat Bot

Task 2.1: Script tự động hóa cấu trúc Folder Drive

Tự động tạo Folder môn học và 8 Folder Lab khi có lớp mới.

Task 2.2: Cấu hình Google Chat API

Kết nối Apps Script với Google Chat để nhận lệnh điều khiển.

Task 2.3: Phát triển lệnh /lab cho Bot

Bot nhận link bài nộp, kiểm tra quyền và đổi tên file đúng chuẩn PSxxxx.

Task 2.4: Viết Script quét bài nộp tự động

Quét Drive và cập nhật trạng thái "Đã nộp" vào Sheet TienDo_Lab_Quiz.

Sprint 3

Giai đoạn 3: Điểm danh GPS & Nhắc nhở

Task 3.1: Phát triển lệnh /checkin (GPS)

Tính toán khoảng cách Haversine giữa SV và GV để điểm danh.

Task 3.2: Xây dựng Time-trigger Reminder

Tự động gửi Email/Chat nhắc nhở SV chưa nộp bài vào 22:00 hàng ngày.

Sprint 4

Giai đoạn 4: Quiz Sync & Hoàn thiện

Task 4.1: Viết Chrome Extension lấy điểm Quiz

Bóc tách dữ liệu từ ILIAS và đẩy trực tiếp về hệ thống Sheet.

Task 4.2: Tối ưu hóa AppSheet Dashboard

Thiết kế biểu đồ Real-time cho báo cáo tiến độ tổng thể.

Task 4.3: Kiểm thử Beta trên lớp thực tế

Vận hành thử nghiệm và tinh chỉnh lỗi hệ thống.
